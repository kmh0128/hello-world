변수
===

"변수란, 단 하나의 값을 저장할 수 있는 메모리 공간"


변수의 선언과 초기화
===

[접근제어자] 타입 변수명

접근제어자: 변수의 접근범위를 지정

타입: 자료형으로 자바는 명시적으로 타입을 지정해야 함.

ex)예시

int age

int가 변수 타입

age가 변수명이다.

위에것을 정리하면, 

변수타입: 변수에 저장될 값이 어떤 '타입'인지 지정 (ex. int(정수))

변수이름: 변수에 붙인 이름
